# dotfiles

## One-Time Setup

```
cd ~ && \
git clone git@github.com:zwbetz-gh/dotfiles.git && \
cd dotfiles && \
./setup.sh
```

## Bash Function Library

Available functions can be viewed with:

```
compgen -A function | grep 'bfl_'
```

Or, by typing `bfl_` and doing a trusty old `Tab` + `Tab`
